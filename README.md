# Iris Flower Classification

## CodeAlpha DATA SCIENCE Internship - Task 1

### Project Overview

This project aims to classify Iris flowers into three different species using Machine Learning.

The three species are:

* Iris Setosa
* Iris Versicolor
* Iris Virginica

The classification is performed based on the following flower measurements:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

### Dataset

Dataset used: Iris Dataset

Source:
https://www.kaggle.com/datasets/saurabh00007/iriscsv/data
### Technologies Used

* Python
* Pandas
* Scikit-Learn
* Machine Learning

### Algorithm Used

K-Nearest Neighbors (KNN)

### Project Workflow

1. Load the Iris Dataset
2. Data Preprocessing
3. Split Dataset into Training and Testing Sets
4. Train the KNN Model
5. Make Predictions
6. Evaluate Model Performance

### Results

The model successfully classified Iris flower species with high accuracy.
